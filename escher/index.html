1. Abstract

The escher is a unary operator for discrete logarithmic involutions. It defines two distinguished values that are orthogonal to the rational number line.

---
2. Elements

Escher operates on signed 1, α, and ω, with any rational coefficient n.

With alpha and omega being multiplicative inverses:

α ^ –1 = ω
---
3. The Escher Operator

The escher operator, denoted Ε, is an operational involution between two orthogonal projections: –1, 1 and α, ω. It's a dual operator, with both alpha and omega variants.

Every element is indexed by a rational parameter n and admits two projections: scalar form n and exponential form α ^ n .

3.1 Alpha Escher Operator

The alpha escher operator flips ones with alphas, inverse ones with omegas.

Ε _ α ( n ) = α ^ n
Ε _ α ( α ^ n ) = n
3.1 Omega Escher Operator

The omega escher operator is the reciprocal image of the alpha flip.

Ε _ ω ( n ) = ω ^ n
Ε _ ω ( α ^ n ) = n
3.1 Escher Duality

For all terms t:

Ε _ α ( –t ) = Ε _ ω ( t )
3.1 Logarithmic Escher

Introduce a "variant" index v ∈ { α, ω }. Then the Escher operator can be written as

Ε _ v ( x ) .

The Escher operator is a logarithm involution.

Ε _ v ( a · b ) = Ε _ v ( a ) + Ε _ v ( b )
Ε _ v ( a + b ) = Ε _ v ( a ) · Ε _ v ( b )
Ε _ v ( a ^ b ) = b · Ε _ v ( a )
