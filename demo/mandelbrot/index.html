Rendering Mandelbrot

using Constructive Operational Type Theory

The Mandelbrot fractal follows the iteration rule:

z_{n+1} = z_{n}^2 + c

Classic complex numbers assume a dyadic basis, because i = sqrt(-1)

x + y*i

Translating this to the zero-lift, we get our basis:

x + y*0^{w/2}

Then we square it.

(x + y*0^{w/2})^2
= x^2 + 2xy*0^{w/2} - y^{2}

And finally, determine z for the first iteration:

z_{1} = (x^2 + 2xy*0^{w/2} - y^{2}) + (x + y*0^{w/2})
z_{1} = x^2 + x + 2xy*0^{w/2} + y*0^{w/2} - y^{2}

Let's try a sample value, like (0.25, 0.5):

z_{1} = (0.25)^2 + (0.25) + 2*(0.25)*(0.5)*0^{w/2} + (0.5)*0^{w/2} - (0.5)^{2}
z_{1} = 0.0625 + 0.75*0^{w/2}
