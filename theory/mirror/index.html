2. Signature

We define the formal language of mirror terms.

2.1 Atomic Terms

The following are atomic terms:

  • The distinguished element 0
  • The distinguished element ω

These symbols are primitive elements of the language.

2.2 Function Symbols
  • A unary function symbol 𝕊 (successor)
2.3 Formation of Terms

  • Every atomic term is a term.

  • If a is a term, then so is:

    −a

  • If a and b are terms, then the following are also terms:

    𝕊^a(b)
    a+b
    a·b
    a^b

Exponentiation is a primitive syntactic operation; at this stage, no semantic restrictions are imposed.

2.4 Notation
For each finite natural number n, we write:
n:=𝕊^n(0)
as a notational convenience.

5. Derived Properties
5.1 Derivation Semantics

All logarithmic and exponential identities are consequences of the single reciprocity axiom α·ω=1, combined with the inductive term structure and primitive operations.

v^x = y

log_v y = x

4.2 Reciprocity-Driven Mapping
α^(-n ) = ω^n
log_α( ω^n ) = -n
α^( α^n ) = n
log_α( n ) = α^n
α^(ω^n ) = -n
log_α( -n ) = ω^n
ω^(-n ) = α^n
log_α( ω^n ) = -n
α^(ω^n ) = -n
log_α( -n ) = ω^n
4.2 Rewrite Rules
log_v ( x · y ) = log_v ( x ) + log_v ( y )
log_v ( x + y ) = log_v ( x ) · log_v ( y )

7. Derivations
7.1 Reciprocity

The reciprocity axiom:

α · ω = 1

Reciprocal rule of the exponential:

α^-v = ω^(v )
α^-n = ω^(n )

Reciprocal rule for logarithms:

log_v( α ) = – log_v( ω )
log_v( ω ) = – log_v( α )
log_v( n ) = – log_v( 1 / n )
5.2 Involution

The involution axiom:

α^( α^n ) = n

Forces:

log_α( n ) = α^n

Substitute the reciprocal in the exponent:

α^( ω^–n ) = n

Take the logarithm:

log_α( α^( ω^–n ) ) = log_α n
ω^–n · log_α( α ) = log_α n
