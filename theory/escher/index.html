2. Signature

We define the formal language of Escher terms.

2.1 Atomic Terms

The following are atomic terms:

  1. Every rational number

    n ∈ ℚ

  2. The distinguished constants

    v ∈ { α, ω }

These symbols are primitive elements of the language.

2.2 Formation of Terms

The set of terms is defined inductively:

  1. Every atomic term is a term.

  2. If a and b are terms, then so are

    a + b
    a · b
    a ^ b

Exponentiation is a primitive syntactic operation; no restrictions on term types are imposed at this stage.

4. Derived Properties
4.1 Derivation Semantics

All logarithmic and exponential identities are consequences of the single reciprocity axiom α · ω = 1, combined with the inductive term structure and primitive operations.

v^x = y

log_v y = x

4.2 Reciprocity-Driven Mapping
α^(-n ) = ω^n
log_α( ω^n ) = -n
α^( α^n ) = n
log_α( n ) = α^n
α^(ω^n ) = -n
log_α( -n ) = ω^n
ω^(-n ) = α^n
log_α( ω^n ) = -n
α^(ω^n ) = -n
log_α( -n ) = ω^n
4.2 Rewrite Rules
log_v ( x · y ) = log_v ( x ) + log_v ( y )
log_v ( x + y ) = log_v ( x ) · log_v ( y )

5. Derivations
5.1 Reciprocity

The reciprocity axiom:

α · ω = 1

Reciprocal rule of the exponential:

α^-v = ω^(v )
α^-n = ω^(n )

Reciprocal rule for logarithms:

log_v( α ) = – log_v( ω )
log_v( ω ) = – log_v( α )
log_v( n ) = – log_v( 1 / n )
5.2 Involution

The involution axiom:

α^( α^n ) = n

Forces:

log_α( n ) = α^n

Substitute the reciprocal in the exponent:

α^( ω^–n ) = n

Take the logarithm:

log_α( α^( ω^–n ) ) = log_α n
ω^–n · log_α( α ) = log_α n
